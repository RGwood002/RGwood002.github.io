Sun Tracking Project

The suntracking project was done with a friend. The goal was to display the suns location on text through an app. I was responsible for the embedded side. I wired two ESP8266 boards to TMP36 sensors, had the data sent to an AWS EC2 server that had a LAMP stack installed, and saved the data to a database.

Fridge Mate

The Fridge Mate project was to help users manage the contents of their fridge. With a Raspberry Pi we could sent information on food scanned to a database and have it displayed to the user on an app. I developed the app using Android Studio and Java. The app would remind users when food was about to expire, the user could manually add foods if it couldn't be scanned as well as delete foods, and sort the foods based on when they were scanned or what was about to expire
